SkyFightPlus

SkyFightPlus is an interactive multimedia player that lets you play songs and ads in a customizable playlist. Built with HTML, CSS, and JavaScript, it allows users to:

Add songs or ads (audio or video).

Shuffle songs and automatically play ads at set intervals.

Select/unselect items in the playlist for easy deletion.

Dynamically change the ad interval.

Enjoy a clean, neon-green themed main menu with intuitive controls.

Watch video ads directly in the player with proper playback handling.

This project is perfect for anyone who wants a customizable music/advertisement player with a visually appealing and interactive interface.

Features:

Add multiple songs and ads easily.

Automatically resumes songs after ads finish.

Countdown timer shows when the next ad will play.

Modern neon theme for a futuristic look.

Full video and audio support.

User-friendly playlist with selection highlights and delete functionality.

Usage:

Open the website in your browser.

Use the buttons in the main menu to add songs or ads.

Click “Start Player” to begin playback.

Change the ad interval using the input box if desired.

Select items in the playlist to delete them****
